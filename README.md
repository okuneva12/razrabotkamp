# WeatherWise - Концепция мобильного приложения
1. Название и краткое описание
WeatherWise - простое мобильное приложение для просмотра текущей погоды и прогноза. Приложение показывает погоду в текущем местоположении пользователя и позволяет добавлять избранные города.

2. Основные функции (MVP)
Просмотр текущей погоды в вашем местоположении
Добавление/удаление избранных городов
Просмотр прогноза погоды на 3 дня
Базовая информация: температура, влажность, скорость ветра, иконка погоды

5. Сценарии использования
Сценарий 1: Просмотр текущей погоды
Пользователь открывает приложение
Приложение автоматически определяет местоположение
Отображается текущая погода и прогноз
Пользователь может добавить город в избранное

Сценарий 2: Добавление города в избранное
Пользователь нажимает "Перейти к избранным"
Открывается список избранных городов
Пользователь нажимает "Добавить новый город"
Вводит название города в поиске
Выбирает нужный город из результатов
Город добавляется в список избранных

Сценарий 3: Просмотр погоды в избранном городе
Пользователь открывает список избранных городов
Нажимает на любой город в списке
Открывается главный экран с погодой для выбранного города

6. Чек-лист функционала по лабораторным работам
ЛР4 (Верстка)
Создать главный экран с отображением погоды
Создать экран избранных городов
Создать экран добавления города
Реализовать навигацию между экранами
Добавить базовые стили и цветовую схему (синие тона для погоды)

ЛР5 (Логика)
Реализовать обработку ввода названия города
Добавить логику добавления/удаления избранных городов
Реализовать переключение между городами
Добавить валидацию ввода
Реализовать базовые анимации переходов

ЛР6 (Данные)
Интеграция с OpenWeatherMap API (https://openweathermap.org/api)
Хранение избранных городов в SharedPreferences
Кэширование данных о погоде
Обработка ошибок сети
Автоматическое обновление данных о погоде

7. API для использования
OpenWeatherMap API (Free tier):
Текущая погода: https://api.openweathermap.org/data/2.5/weather
Прогноз на 5 дней: https://api.openweathermap.org/data/2.5/forecast
Поиск городов: https://api.openweathermap.org/geo/1.0/direct

8. Технические требования
Целевая платформа: Android
Минимальная версия API: 21 (Android 5.0)
Язык разработки: Kotlin
Архитектура: MVVM
Библиотеки: Retrofit для API, Room для базы данных
